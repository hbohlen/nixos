# WiFi Troubleshooting Quick Reference

## Problem: WiFi Password Prompts After Every Reboot

**Root Cause:** NetworkManager connection files are not persisted due to impermanence configuration issues.

**Quick Fix:**
```bash
# Use automated diagnostics and repair
./scripts/wifi-diagnostics.sh --repair

# Select option 4 (All fixes) or option 1 (Fix impermanence persistence)
```

**Manual Fix:**
```bash
# 1. Ensure persistence directory exists
sudo mkdir -p /persist/etc/NetworkManager/system-connections

# 2. Copy current connections
sudo cp /etc/NetworkManager/system-connections/* /persist/etc/NetworkManager/system-connections/

# 3. Fix permissions
sudo chown root:root /persist/etc/NetworkManager/system-connections/*
sudo chmod 600 /persist/etc/NetworkManager/system-connections/*

# 4. Rebuild system
sudo nixos-rebuild switch --flake .#laptop
```

## Problem: Frequent WiFi Disconnections

**Causes:**
- MAC address randomization compatibility issues
- Aggressive power management
- Driver issues

**Solutions:**
```bash
# Disable MAC randomization for all WiFi connections
nmcli connection show | grep wifi | while read line; do
    conn_name=$(echo "$line" | awk '{print $1}')
    nmcli connection modify "$conn_name" wifi.mac-address-randomization no
done

# Disable WiFi power saving
sudo iwconfig wlan0 power off

# Check TLP settings
tlp-stat -s | grep -i wifi
```

## Problem: Can't Discover WiFi Networks

**Solutions:**
```bash
# Check if WiFi is blocked
rfkill list
sudo rfkill unblock wifi

# Restart WiFi interface
sudo ip link set wlan0 down
sudo ip link set wlan0 up

# Reload WiFi drivers
sudo modprobe -r iwlwifi && sudo modprobe iwlwifi

# Restart NetworkManager
sudo systemctl restart NetworkManager
```

**Connection Details:**
```bash

# Show specific connection settings
nmcli connection show "WiFi-Name"

# Check MAC randomization setting
nmcli connection show "WiFi-Name" | grep mac-address-randomization
```

## Configuration Files

**NetworkManager Main Config:**
- Location: `/etc/NetworkManager/NetworkManager.conf`
- Generated by NixOS WiFi module

**Connection Files:**
- Active: `/etc/NetworkManager/system-connections/`  
- Persistent: `/persist/etc/NetworkManager/system-connections/`
- Required permissions: `600 root:root`

**NixOS WiFi Module:**
- Location: `modules/nixos/wifi.nix`
- Settings: Power saving, MAC randomization, firmware

## Emergency Recovery

**If WiFi stops working completely:**
```bash
# 1. Reset NetworkManager
sudo systemctl stop NetworkManager
sudo rm -rf /var/lib/NetworkManager/*
sudo systemctl start NetworkManager

# 2. Reload network stack
sudo systemctl restart systemd-networkd
sudo systemctl restart NetworkManager

# 3. Reset WiFi interface
sudo ip link set wlan0 down
sudo modprobe -r iwlwifi
sudo modprobe iwlwifi
sudo ip link set wlan0 up
```

**If impermanence is broken:**
```bash
# Manual connection creation
nmcli device wifi connect "SSID" password "password"

# Copy to persistence after connection works
sudo mkdir -p /persist/etc/NetworkManager/system-connections
sudo cp /etc/NetworkManager/system-connections/* /persist/etc/NetworkManager/system-connections/
sudo chown root:root /persist/etc/NetworkManager/system-connections/*
sudo chmod 600 /persist/etc/NetworkManager/system-connections/*
```

## Prevention

**NixOS Configuration Checklist:**
- [ ] WiFi module enabled: `wifi.enable = true`
- [ ] Firmware enabled: `wifi.enableFirmware = true`
- [ ] Impermanence includes: `/etc/NetworkManager/system-connections`
- [ ] wpa_supplicant disabled: `networking.wireless.enable = false`
- [ ] Power saving configured: `wifi.powerSaving = "low"`

**Regular Maintenance:**
```bash
# Weekly: Check system integrity
./scripts/wifi-diagnostics.sh

# After updates: Verify configuration
sudo nixos-rebuild switch --flake .#laptop
nmcli connection show
```
