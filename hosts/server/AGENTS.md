# AGENTS.md

## Directory Purpose
This directory contains the server host configuration, optimized for headless operation, service hosting, and minimal resource usage.

## Files in This Directory
- `default.nix` - Main server system configuration focused on services, security, and minimal desktop components
- `hardware-configuration.nix` - Hardware-specific configuration generated by nixos-generate-config for this server machine

## Dependencies
- Imports common system modules from `/modules/nixos/common.nix`
- Imports ZFS and disk configuration from `./hardware/disko-zfs.nix` (host-specific)
- Imports impermanence configuration from `/modules/nixos/impermanence.nix`
- May have minimal or no Home Manager integration compared to desktop/laptop hosts
- Uses root-level flake.nix for system definition

## Notes for AI Agents
- Server configuration should prioritize stability, security, and resource efficiency
- Minimize GUI components and desktop applications
- Focus on system services, networking, and server-specific software
- Ensure proper firewall configuration and security hardening
- Consider automated updates and monitoring for server environments
- Test service configurations thoroughly as servers often run critical applications
